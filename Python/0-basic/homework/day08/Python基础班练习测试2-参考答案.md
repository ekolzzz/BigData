#### 18． 根据下面要求完成程序设计？

已知列表`alpha_list = ['A', 'B', 'C', 'A', 'B', 'B']`，统计列表中元素的出现的频次？

预期输出：

```bash
字母:A 频次: 2
字母:B 频次: 3
字母:C 频次: 1
```

```python
'''代码在PyCharm写完，保证能够运行后直接拷贝到这里'''
alpha_list = ['A', 'B', 'C', 'A', 'B', 'B']

data = {}
for i in alpha_list:    
    data[i] = alpha_list.count(i)
    
for k, v in data.items():    
    print('字母:%s 频次: %d' % (k, v))
```
